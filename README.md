# 🌦️ Weather Forecast Android App

A simple and modern **Android Weather Forecast Application** that displays real-time weather information such as temperature, humidity, wind speed, and weather conditions for a given city.

Built using **Kotlin, Android Studio, and Weather API**, the app demonstrates API integration, clean UI design, and Android development practices.

---

## 📱 Features

- 🌍 Search weather by city
- 🌡️ Display real-time temperature
- 💧 Show humidity levels
- 🌬️ Display wind speed
- ☁️ Weather condition indicators
- 📊 Clean and user-friendly interface

---

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **IDE:** Android Studio  
- **API Integration:** Retrofit  
- **Architecture:** MVVM  
- **UI:** XML Layouts  
- **Build System:** Gradle

## 🚀 How to Run the Project

1. Clone the repository


git clone https://github.com/Soumo666x/weather-forecast-android-app.git


2. Open the project in **Android Studio**

3. Sync Gradle

4. Run the application on:
   - Android Emulator
   - Physical Android Device
